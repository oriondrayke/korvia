# Korvia Design System

A living, versioned design system for web projects.

## Live site

https://oriondrayke.github.io/korvia/

## What's inside

- `korvia.css` — design tokens and utility classes
- `korvia.js` — base interactions
- `index.html` — system overview
- `components/` — individual component pages *(coming)*
- `examples/` — example page layouts *(coming)*

## How to use

Link to the CSS in your HTML:

```html
<link rel="stylesheet" href="https://oriondrayke.github.io/korvia/korvia.css">
```

Or copy `korvia.css` into your project.

## Status

Live design system — updates pushed to GitHub reflect immediately on the site.
